# LeiloesTDSat

Sistema desenvolvido para uma casa de leilões, com o objetivo de informatizar e organizar as rotinas de cadastro e listagem de produtos.

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos cadastrados
- Integração com banco de dados MySQL

## Tecnologias Utilizadas

- Java
- NetBeans
- MySQL
- Git
- GitHub
